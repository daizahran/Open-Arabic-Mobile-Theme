
 خطوات التثبت :
----------------

1. قم بتثبيت اضافة "WPtouch" من لوحة التحكم المدونة او من الموقع التالي : http://wordpress.org/extend/plugins/wptouch
2. قم باستبدال محتويات مجلد الثيم الافتراضي للاضافة wp-content/plugins/wptouch/themes/default بمحتويات مجلد الثيم هذا

ملاحظة: يمكنك استبادل ملف favicon.png بايقونة موقعك.

 Installation Steps :
----------------------

1. Install "WPtouch iPhone Theme" plugin : http://wordpress.org/extend/plugins/wptouch
2. Replace default directory of the plugin on relative path 'wp-content/plugins/wptouch/themes/default' with this directory

Note: you can replace favicon.png with favicon of your website.
