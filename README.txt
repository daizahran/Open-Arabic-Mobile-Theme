
 خطوات التثبت :
---------------

1. انسخ مجلد الثيم الى مجلد الثيمات في المدونة على المسار التالي : wp-content/themes
2. قم بتثبيت اضافة "WordPress Mobile Pack" من الموقع التالي : http://wordpress.org/extend/plugins/wordpress-mobile-pack
3. انتقل الى قائمة المظهر (Appearance) من لوحة التحكم ثم اختر "Mobile Switcher".
4. اختر "Open Arabic Mobile Theme" من قائمة "Mobile Theme" ليكون ثيم الهواتف الذكية في المدونة.

ملاحظة: يمكنك استبادل favicon.png بايقونة موقعك.


 Installation Steps :
---------------------

1. Copy folder of the theme to the following relative path : wp-content/themes
2. Install "WordPress Mobile Pack" plugin : http://wordpress.org/extend/plugins/wordpress-mobile-pack
3. Go to "Appearance" menu of the control panel then select "Mobile Switcher".
4. Set "Open Arabic Mobile Theme" as "Mobile Theme".

Note: you can replace favicon.png with favicon of your website.
